#  UNO P2P Multiplayer

A professional, offline-first, Peer-to-Peer (P2P) Multiplayer Uno card game. Built with clean, responsive UI and modern web technologies.

## 🚀 Features
- **P2P Multiplayer:** Play in real-time with friends using PeerJS, no backend server required.
- **Offline-First:** Built as a Progressive Web App (PWA) with offline capabilities.
- **Premium UI:** Sleek dark-mode interface with card-fan animations and responsive layout.
- **Chat System:** Real-time chat integration for online sessions.
- **Cross-Platform:** Works seamlessly on mobile and desktop browsers.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Networking:** PeerJS (for real-time P2P communication)
- **Architecture:** Progressive Web App (PWA)
